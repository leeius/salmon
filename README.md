# Salmon Innovations

React and Vite project scaffold for Salmon Innovations.

## Composition

- Navbar
- Hero
- About
- Services
- Projects
- Footer

## Routes

- `/` Home

## Stack

- React 19
- TypeScript
- Vite
- react-router-dom
- Tailwind CSS
- framer-motion
- react-icons
- axios

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.

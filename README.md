# Salmon Innovations

React and Vite project scaffold for Salmon Innovations.

## Composition

- Navbar
- Hero
- About
- Services
- Projects
- CTA
- Footer

## Routes

- `/` Home
- `/about` About
- `/services` Services
- `/work` Our Work
- `/careers` Careers
- `/contact` Contact

## Stack

- React 19
- TypeScript
- Vite
- react-router-dom
- Tailwind CSS
- framer-motion
- react-icons
- axios

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
